# 贡献指南

v2.0 的首要规则是：任何主题功能都不能改变 ChatGPT/Codex 的启动、应用包、缓存、登录状态或对话数据。

## 可以贡献

- 调整 `themes/*/palette.json` 中的官方外观配色；
- 改进预览图、说明和无障碍对比度；
- 完善安全中心的只读检测与诊断；
- 让旧版残留清理更精确、更易审计；
- 改进安装包构建、哈希和测试。

## 禁止进入主分支

- CDP、远程调试端口或渲染器注入；
- 自动重启 ChatGPT/Codex；
- 后台 watcher 或登录启动项；
- 修改、重置、移除或重新注册官方应用包；
- 删除或搬移官方应用缓存；
- 删除或读取登录凭据内容；
- 自动写入未公开的内部配置键；
- 修改官方签名文件或 `app.asar`。

## 提交前检查

```bash
node tests/test-palette.mjs
node tests/test-windows-safety.mjs
```

Windows 脚本还必须通过 PowerShell AST 解析。涉及清理目标的修改，需要在 PR 中逐项说明路径、删除理由、保护范围和失败回滚方式。
